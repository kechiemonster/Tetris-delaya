Tetris voor Delaya
===================

Bestanden:
- index.html
- style.css
- tetris.js
- bg.svg (achtergrond)
- music.wav (kort, loopbaar muziekje)

Hoe te gebruiken:
1. Pak de ZIP uit.
2. Upload alle bestanden naar jouw GitHub repository (bijvoorbeeld in de root of in een map 'tetris').
3. Als je GitHub Pages wilt gebruiken, zet de branch 'main' of 'gh-pages' aan in de repo instellingen en stel de root of map als source.
4. Open de GitHub Pages link (of open index.html direct op je telefoon) om het spel te spelen.

Opmerking:
- Mobiele browsers vereisen soms een aanraking voordat audio start. Als de muziek niet automatisch begint, tik dan op "Start spel" knop (die start ook de muziek).
